# Flarum 文档

本文档翻译自 [Flarum 官方文档](http://flarum.org/docs/)。

Flarum 是一款现代、优雅、简洁、强大的论坛软件。Flarum 让在线交流变得更加轻松愉快。

[开始阅读](http://flarum.org.cn/docs/)

本手册由 [Flarum 中文社区](http://discuss.flarum.org.cn)翻译。
